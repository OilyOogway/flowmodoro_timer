/*******************************************************************************

* README
* Author: Luke Tran
*******************************************************************************/

Program Purpose:
---------------


   This extension is a minimalistic implementation of the flowmodoro technique
   used to maintain focus. It automatically calculates the time needed for each
   break.


Compile/run:
-----------


   Clone and run by loading into chrome://extensions
